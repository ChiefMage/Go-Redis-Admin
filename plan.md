## Redis Admin系统功能分析

####  管理员模块

	1. 管理员登录
	2. 管理员密码修改  

#### Redis 配置模块

1. redis 配置信息填写(redis实例名称、IP、端口、账号、密码等)
2. redis链接测试
3. redis配置状态管理（启用、禁用）

#### Redis 前缀配置模块

1. 前缀的添加（选择redis实例）

   例如：用户信息缓存的key: userinfo_123456,设置keyname =用户信息,keyvalue = userinfo,便于检索

2. 前缀的修改、删除等功能
